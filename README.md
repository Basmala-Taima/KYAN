# كَيان | KYAN 🌱
**"لأن القوة تبدأ من الداخل" | "Because strength starts from within"**

منصة تفاعلية للدعم النفسي والتمكين، تهدف لخلق مساحة آمنة وملهمة للجميع.

A psychological support and empowerment platform designed to be a safe, inspiring space for everyone.

---
### 🔗 [Live Demo | تجربة الموقع من هنا](https://basmala-taima.github.io/KYAN/)
---

## ✨ المميزات | Features

* 🌍 **Bilingual Support:** دعم كامل للغتين العربية والإنجليزية.
* 🧘 **Strength Steps:** تمارين تفاعلية (ذهنية، جسدية، روحية).
* 🌊 **Tranquility Harbors:** أدوات هادئة لتعزيز السلام الداخلي.
* 📝 **Safe Space:** مفكرة خاصة لحفظ تدويناتك بخصوصية تامة (Local Storage).
* 📱 **Fully Responsive:** تصميم متجاوب تماماً مع جميع الشاشات باستخدام Tailwind CSS.

---

## 🛠 التكنولوجيا المستخدمة | Tech Stack
* **HTML5** & **CSS3** & **Tailwind CSS**
* **JavaScript (ES6+)**
* **FontAwesome Icons**
* **Google Fonts (Cairo & Outfit)**

---

## 🚀 تشغيل المشروع | How to Run
1. Clone the repository.
2. Open `index.html` in your browser.
