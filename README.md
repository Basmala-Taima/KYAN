## ✨ المميزات | Features

* 🌍 **Bilingual Support:** دعم كامل للغتين العربية والإنجليزية.
* 🧘 **Strength Steps:** تمارين تفاعلية (ذهنية، جسدية، روحية).
* 🌊 **Tranquility Harbors:** أدوات هادئة لتعزيز السلام الداخلي.
* 📝 **Safe Space:** مفكرة خاصة لحفظ تدويناتك بخصوصية تامة (Local Storage).
* 📱 **Fully Responsive:** تصميم متجاوب تماماً مع جميع الشاشات باستخدام Tailwind CSS.

---

## 🛠 التكنولوجيا المستخدمة | Tech Stack

* **JavaScript (ES6+)**
* **FontAwesome Icons**
* **Google Fonts (Cairo & Outfit)**
